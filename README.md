## 🛠️ Installation Steps:

1. Clone the repository

```bash
git clone https://github.com/vivek80801/studentApp.git
```

2. Change the working directory

```bash
cd studentApp
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

### 🌟 You are all set!
